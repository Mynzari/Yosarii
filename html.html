Utiliza la etiqueta ul para crear listas y estas otras dos etiquetas son necesarias igual, il para listas desordenadas y ul para listas enumeradas y más ordenadas

Puedes utilizar la etiqueta img para adjuntar imágenes, esta etiqueta tiene ciertos atributos pero no es muy recomendable usarlos, son height y width, height define la altura de la imagen y width el ancho.
